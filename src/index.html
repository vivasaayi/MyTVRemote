Sony TV Remote


Set a Pre-Shared Key on your TV under Network > Home Network Setup > Remote Device Settings, then enter it here. For PIN pairing, leave PSK blank and enter the PIN shown on TV after first command attempt.

Android TV Remote Control (Unavailable)

⚠️ Remote control functionality is currently unavailable. The required Android TV Remote library could not be loaded.

Use IRCC commands with PSK or PIN authentication instead. Buttons remain disabled to avoid confusion.

Android TV Pairing (Scalar API)

Start pairing to make the TV display a PIN, then enter that PIN below to register this app.

After a successful pairing, save the Client ID and User ID for authenticated API calls.
